# Yuyu Storefront

A polished, responsive e-commerce frontend for Yuyu Online Shopping built with React, htm, Tailwind CDN, hash routing, dark mode, shared state, wishlist, cart, loading skeletons, and toast notifications.

---

## Project Overview

Yuyu Storefront is a modern e-commerce frontend designed to deliver a clean, fast, and responsive shopping experience. It follows a minimalist UI approach inspired by modern platforms like Apple and Shopify, focusing on usability, smooth interactions, and scalability.

The project uses reusable React components, mock product data, and a modular structure to simulate a real-world online shopping system.

---

## Features

* Responsive design (mobile + desktop)
* Dark mode support
* Sticky navbar with logo, search bar, cart icon, and user profile
* Hero section with promotional banner
* Category system with icons/images
* Product grid with filtering
* Cart and checkout UI
* Login / Signup pages
* Wishlist support
* Toast notifications
* Loading skeletons
* Smooth hover animations and transitions
